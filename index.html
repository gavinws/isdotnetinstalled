Is .NET Framework installed?

And what versions?
download .ZIPdownload .TGZ

Try one of the following to find out! A list of installed versions will be displayed.

Paste the following in a PowerShell prompt and press enter twice to run:

Get-ChildItem 'HKLM:\SOFTWARE\Microsoft\NET Framework Setup\NDP' -recurse |
Get-ItemProperty -name Version, release -EA 0 | 
Where { $_.PSChildName -match '^(?!S)\p{L}'} |
Select PSChildName, Version, Release,`
@{Label=".NET 4.5.1+";Expression={"{0:N}" -f ($_.Release -gt 378389)}} |
Format-Table -AutoSize

Or download and run this totally safe .NET 2.0 executable which will do essentially the same thing as the PowerShell script above: DotNetVersionList.exe (VirusTotal Results)
